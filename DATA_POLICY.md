# Data Policy & Privacy Guidance

This repository contains code and educational artifacts. Any included survey-derived data must be treated as **sensitive educational research data**.

## 1. Scope
Applies to all raw response files, intermediate cleaned datasets, exported summaries, figures, and sentiment outputs derived from the Physics Wallah Student Experience survey.

## 2. Allowed Usage
- Local analysis for learning and method demonstration.
- Academic review of methodology.
- Non-commercial reproducibility exercises.

## 3. Prohibited Usage
- Re-publication of full datasets.
- Commercial exploitation.
- Attempting to deanonymize or re-identify respondents.
- Merging with external personally identifying datasets.

## 4. Handling Guidelines
- Keep raw data files out of version control. Use `.gitignore` to exclude them.
- Never push files containing emails, names, phone numbers, addresses, or IDs.
- Limit distribution to aggregated statistics and anonymized charts.
- When sharing screenshots/figures, ensure small categories do not expose individuals.

## 5. Removal / Takedown
If you find any file that may contain sensitive or identifying information:
1. Stop further distribution.
2. Open a GitHub issue marked `[PRIVACY]` or email the maintainer.
3. The maintainer will remove or redact the file promptly.

## 6. Disclaimer
The maintainer provides the code and educational structure AS-IS and does not guarantee perfect anonymization. Users cloning or forking the project accept responsibility for proper data handling.

## 7. Contact
Open an issue in this repository for clarification or permission requests.
